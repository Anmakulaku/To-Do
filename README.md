# Drag and drop ToDo App

This is a simple Todo application with drag and drop functionality built using React and TypeScript. The application allows users to create, edit, and delete tasks while also providing the ability to rearrange tasks through drag and drop.

# Live

https://anmakulaku.github.io/To-Do/

# Tech used
<ul>
  <li>React</li>
  <li>TypeScript</li>
  <li>Vanilla CSS</li>
  <li>React-beautiful-dnd</li>
</ul>
